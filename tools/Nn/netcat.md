##### Usage:
`nc [options] [hostname] [port]`        
`nc -l [options] [port]`

> -l
>> Enables listening mode, allowing the tool to wait for incoming connections.

> -v
>> Produces more detailed output. Use it twice for extra verbosity.

> -n
>> Uses numeric IP addresses only, skipping DNS lookups.

> -p
>> Sets the local port to use. Accepts a single port or a range (lo–hi).
